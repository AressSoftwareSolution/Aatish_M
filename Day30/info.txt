UI Updates Product List



4️⃣ Example
In UI

You type:

Name: Apple
Quantity: 10
Price: 20

UI sends request:

POST /products

Body:

{
 "name": "Apple",
 "quantity": 10,
 "price": 20
}

FastAPI stores it and returns response.

Then UI calls:

GET /products

to display list.   



=======process to run code=======
uvicorn fast_api:app --reload   ---- run on terminal
